# Calculadora de Media, Varianza y Desviación Típica

Proyecto en Python para calcular media, varianza y desviación estándar (typica) de una lista de 9 números con numpy.

- El programa toma exactamente 9 números, los organiza en una matriz de 3x3 y devuelve las estadísticas por filas, columnas y total.
- Incluye tests automáticos y un ejemplo de uso simple.
- Minimalista, sin complicaciones.

## ¿Cómo usar?

1. Instala las dependencias:  
   `pip install -r requirements.txt`

2. Ejecuta el programa:  
   `python main.py`

El código principal está en `mean_var_std.py`.  
Puede ser útil para análisis rápidos o como referencia educativa.
